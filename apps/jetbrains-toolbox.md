# JETBRAINS-TOOLBOX

 Essential tools for software developers and teams.
 
 NOTE: this AppImage will automatically create launchers in:
 - ~/.config/autostart
 - ~/.local/share/applications
 
 The option "-H" of AM/AppMan doesn't work with this AppImage.
